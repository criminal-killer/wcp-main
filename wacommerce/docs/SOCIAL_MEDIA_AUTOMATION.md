# Social Media Automation — SELLA

## The Comment Keyword Strategy

Our growth strategy relies on automated engagement on social media platforms (Facebook, Instagram) to drive traffic to the WhatsApp store.

### Post CTA
"Want to sell on WhatsApp? Comment **READY** 👇 and we'll send you access!"

### Keywords the Bot Watches
- **READY**, Ready, ready
- **SELL**, Sell, sell
- **STORE**, Store, store
- **ME**, Me, me (as in "send me")
- **INTERESTED**, interested
- **YES**, yes, Yes
- **HOW**, how

### Why "READY"?
- **It's action-oriented**: "I'm READY to sell"
- **Works in any language**: Global understanding.
- **Creates urgency**: Implies immediate action.
- **Easy to type**: Low friction for the user.
- **Feels like commitment**: Psychologically primes the user for the next step.

## Automation Workflow
1. **Scrape**: Identify target businesses on OSM/Instagram.
2. **Post**: Share value-driven content (tips, success stories) with the CTA.
3. **Reply**: Bot monitors comments for keywords.
4. **DM**: Bot sends a private message with the waitlist/landing page link.
5. **Convert**: User signs up on the landing page.
