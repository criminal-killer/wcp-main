# Competitor Analysis
# CHATEVO

---

## Direct Competitors

### WATI — wati.io
- **Price:** $99/month (Growth plan)
- **API Markup:** 20% on Meta conversation costs
- **Strengths:** Shopify integration, established brand, team inbox
- **Weaknesses:** Expensive, no in-chat store, no native payments, India-focused
- **Their customers say:** "Too expensive for small businesses" "Basic chatbot" "Markup adds up"

### Interakt — interakt.shop
- **Price:** $55/month
- **API Markup:** 14%
- **Strengths:** Indian market presence, Shopify integration
- **Weaknesses:** Complex UI, basic order management, no in-chat commerce
- **Their customers say:** "Confusing interface" "Limited automation"

### AiSensy — aisensy.com
- **Price:** $49/month
- **API Markup:** 12%
- **Strengths:** Campaign management, good chatbot
- **Weaknesses:** No commerce features at all, messaging-only tool
- **Their customers say:** "Good for campaigns but can't sell through it"

### Zoko — zoko.io
- **Price:** $35/month
- **API Markup:** 15%
- **Strengths:** Partial store integration, Shopify
- **Weaknesses:** Limited store experience, no African payment support
- **Their customers say:** "Decent but half-baked store feature"

### WhatChimp — whatchimp.com
- **Price:** $12-40/month
- **API Markup:** 0%
- **Strengths:** Cheapest, 0% markup
- **Weaknesses:** New, limited features, no commerce
- **Their customers say:** "Good value but basic"

## Chatevo's Position

| Dimension | Competitors | Chatevo |
|-----------|------------|-------|
| Focus | Messaging first | Commerce first |
| In-chat store | None/partial | Full shopping flow |
| In-chat payments | None | M-Pesa, Cards, PayPal |
| Target market | India | Africa → Global |
| Pricing | $35-99/mo | $29/mo |
| API markup | 12-20% | 0% |
| Mini website | None | Auto-generated |
| M-Pesa support | None | Native |
| Revenue dashboard | None | Built-in |

---

## Key Differentiators

1. **Commerce-first:** Built for selling, not just messaging
2. **0% markup:** Customers pay Meta directly
3. **Africa-first:** M-Pesa, mobile money, local languages
4. **Auto mini-website:** Every store gets a free website
5. **In-chat checkout:** Full browse→buy→pay in WhatsApp
6. **Price:** 70% cheaper than market leader
