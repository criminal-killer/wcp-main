# ⚠️ DEPRECATED — Do Not Use

This directory (`wacommerce/phase-2-app`) was an earlier iteration of the  
Chatevo merchant app and is **no longer the canonical source**.

## Canonical Source

| What | Where |
|------|-------|
| **Merchant App** | Repo root (`../../`) — `app/`, `lib/`, `components/` |
| **Admin App** | `wacommerce/admin-panel/` |
| **Workspace entries** | `apps/merchant/`, `apps/admin/` |

## Why this directory exists

This folder is kept for historical reference and Git history preservation.  
It is **not deployed** and should not be referenced in new code.

If Vercel accidentally points at this directory, fix by setting  
**Root Directory = `.`** in the Vercel project settings.
